#  Quiz 1 - Ingenieria Web 

## Generador de mini perfil

El presente proyecto simula una página de registros para estudiantes, donde ellos podrán ingresar su nombre, edad y algun de la carreras dispobles en el sitema ( Civil, Industrial, Sistemas). 
Antes de ingresar los datros se mostrará un mensaje indicando que el perfil se encuentra vacío y esto al llenar los campos requeridos y validar sus respuestas se les informará con otro mensaje si el registro fue exitoso o no.
